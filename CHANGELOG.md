# 更新日志 (Changelog)

## [1.1.0] - 2026-02-04

### ✨ 新增功能 (Features)

- **UI 全面汉化**:
  - 系统所有页面（登录、仪表盘、列表、详情等）均已更新为中文界面。
  - Element Plus 组件库集成中文语言包，日期选择器、分页等组件支持中文。
  - 时间显示格式优化为符合中文习惯（如"刚刚"、"xx分钟前"、"yyyy-MM-dd"）。

- **工作流可视化**:
  - `我发起的文档` 列表中新增工作流进度条。
  - 直观展示当前审批节点（校对/批准）、处理人及状态（不同颜色区分）。
  - 详情页右侧面板增加详细的审批历史记录和状态时间轴。

- **文档预览模式**:
  - 详情页新增「预览/编辑」切换功能。
  - 默认以预览模式（OnlyOffice 只读视图）打开文档，加载速度更快。
  - 只有具备编辑权限的用户可切换至编辑模式进行修改。

- **交互体验优化**:
  - Dashbaord 仪表盘全新设计，统计数据一目了然。
  - 通知点击直接跳转至相关文档详情页。
  - 文档退回操作支持可选填原因，简化流程。

### 🐛 问题修复 (Bug Fixes)

- **OnlyOffice 集成**:
  - 修复了预览模式一直加载中的问题（通过 API `mode` 参数正确控制权限）。
  - 优化 `editorConfig` 配置，开启 `compatibleFeatures` 以提升对复杂 Word 格式（如多级列表、边框）的兼容性。
  - 修复中文文件名上传后的编码显示问题。

- **Docker 网络**:
  - 解决了 Docker 容器内无法访问宿主机 API 的问题。
  - 允许 OnlyOffice 访问私有 IP 地址。

### 🔧 技术调整 (Technical)

- 更新前端 API 封装，支持传递编辑器 view/edit 模式参数。
- 优化后端 `documentController`，在列表接口返回完整的工作流节点信息。
